# 自分について
学生であり研究者であり革命家でもある。  
革新的な科学・技術の調査、プログラムや物づくり、遊びなどを趣味とする。

空間から計算量を取り出して、人類みんなでキャッキャうふふすることが将来の夢。

N次元情報空間 / 0点コンピューティング / 複雑記述記述 が今後の研究テーマ。
